## C++ 基础项目

本项目是用来进行C++之间进行引用的学习文件，主要是要达到以下的目的

1. 实现自定义头文件
2. 实现类的引用
3. 头文件的引用

## 项目结构

```
│  index.cpp               // 主函数
│              
├─class                    // 类文件夹
│      Person.cpp          
│      Person.h
│      
└─utils                    // 工具文件夹
        AgeUtils.cpp
        AgeUtils.h
```
